Configuracion de Acceso Remoto
  • DDNS: Configura port forwarding en tu router (puertos 80, 554, 8000)
  • P2P: Usa la app del fabricante para obtener el ID del dispositivo
  • TURN Server: Para WebRTC remoto, configura un servidor TURN
  • Las URLs remotas necesitan HTTPS para funcionar en navegadores modernos

Configuracion de Streaming

Configura servidores REMOTOS para streaming de camaras desde cualquier lugar.
Conexion Remota Automatica

El sistema detecta automaticamente tu direccion IP/host actual para conectar al servidor WebSocket de camaras.

URL detectada: --

Puerto donde corre tu servidor de streaming (por defecto: 8080)
Usa el boton para aplicar automaticamente la URL detectada, o ingresa una URL personalizada (DDNS, VPS, etc.)
Opciones para Acceso Remoto:
  • Red Local: Usa la URL auto-detectada si estas en la misma red
  • DDNS + Port Forwarding: Configura No-IP, DuckDNS o DynDNS y abre el puerto en tu router
  • VPS: Usa la IP publica o dominio de tu VPS
  • Ngrok/Cloudflare Tunnel: Para exponer sin abrir puertos
Servidores TURN para Acceso Remoto

Los servidores TURN permiten conexiones P2P cuando hay NAT/firewall de por medio.

Ej: turn:turn.example.com:3478 o turn:coturn.example.com:443
Servidores STUN (Gratuitos)
  • stun:stun.l.google.com:19302
  • stun:stun1.l.google.com:19302
  • stun:stun.stunprotocol.org:3478
Opciones de TURN Gratuito/Economico
  • Metered.ca: 500MB gratis/mes (metered.ca)
  • Twilio: $0.40/GB (twilio.com)
  • Coturn: Servidor propio gratuito (github.com/coturn/coturn)

Conectar DVR

Haz clic para iniciar el escaner

Como escanear:

  1. Abre la app de tu DVR (ej: Hik-Connect, gDMSS, XMEye)
  2. Ve a Configuracion > Compartir dispositivo o Codigo QR
  3. Muestra el codigo QR frente a la camara
Variables: {user}, {pass}, {ip}, {port}, {ch} (numero de canal)
Plantillas RTSP por marca:
  • Hikvision: rtsp://user:pass@IP:554/Streaming/Channels/101
  • Dahua: rtsp://user:pass@IP:554/cam/realmonitor?channel=1&subtype=0
  • H.VIEW: rtsp://user:pass@IP:554/stream1
  • XMEye: rtsp://user:pass@IP:554/user=user&password=pass&channel=1
Acceso Remoto al DVR

Conecta tu DVR desde cualquier lugar usando DDNS o servicios P2P del fabricante.

Encuentra esto en la etiqueta del DVR o en la app del fabricante
Requisitos para Acceso Remoto:
  • Configurar Port Forwarding en tu router (puertos 80, 554, 8000)
  • O usar servicio P2P del fabricante (no requiere port forwarding)
  • HTTPS recomendado para conexiones seguras
